LuytenX 汉化修复版
======

这是 [LuytenX](https://github.com/ByteZ1337/LuytenX) 的一个分支，修复了d3d加速导致UI错位的问题并进行了汉化。

Procyon 和 CFR 的反编译器 Gui

根据 Apache 2.0 版许可

## 编译

*****

* 安装[Maven 3](https://maven.apache.org/download.html)
* 克隆此存储库并：`mvn clean install`
